# getDensity

Helper for plotting by density

## Usage

``` r
getDensity(x, y, ...)
```

## Arguments

- x:

  x coordinates

- y:

  y coordinates

- ...:

  Additional parameters to pass to
  [`MASS::kde2d()`](https://rdrr.io/pkg/MASS/man/kde2d.html).

## Value

A matrix of the estimated density.
