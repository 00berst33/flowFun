# plotGroupMFIBars

Function to draw bar plot of MFIs or dMFIs by group.

## Usage

``` r
plotGroupMFIBars(
  input,
  col,
  sample_df,
  comparison,
  populations = NULL,
  transformation = NULL,
  inverse = FALSE,
  upper_lim = NULL
)
```

## Arguments

- input:

  A matrix or data.frame where rows are samples, columns are clusters,
  and values are MFIs for a channel

- col:

  Character, the name of the marker/channel to create the plot for.

- sample_df:

  A factor data frame as generated by
  [`prepareSampleInfo()`](https://00berst33.github.io/flowFun/reference/prepareSampleInfo.md).

- comparison:

  A list defining a comparison.

- populations:

  A vector of strings defining the metaclusters to include in the plot.
  Default is all metaclusters.

- transformation:

  A
  [`transformList`](https://rdrr.io/pkg/flowCore/man/transformList-class.html)
  specifying the transformation applied to the data, if any. Required
  when `input` is a `data.table` and `inverse = TRUE`

- inverse:

  Boolean, whether the data should be back-transformed before plotting.

- upper_lim:

  The y-axis upper limit.

## Value

A bar plot drawn with
[`ggplot2`](https://ggplot2.tidyverse.org/reference/ggplot2-package.html).

## Details

The names and order of the groups defined in `comparison` should appear
in the same way you would like them to appear in the bar plot. Make sure
that the group names you supply are the same as they appear in the
`sample_df` object.
