# makeCountMatrix.FlowSOM

Generate matrix of sample/metacluster cell counts from FlowSOM object.

## Usage

``` r
# S3 method for class 'FlowSOM'
makeCountMatrix(
  input,
  sample_df,
  meta_names = NULL,
  min_cells = 3,
  min_samples = NULL
)
```

## Arguments

- sample_df:

  A data frame containing sample information, generated by
  [`prepareSampleInfo()`](https://00berst33.github.io/flowFun/reference/prepareSampleInfo.md).
